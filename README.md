# 软件清单生成工具

双击 `更新软件清单.vbs` 即可更新软件清单。

## 工作方式

将 `generate_inventory.py` 和 `更新软件清单.vbs` 放在同一文件夹下。双击 `.vbs`，脚本会：

1. 扫描当前文件夹下的 **办公常用** 和 **开发常用** 两个子目录
2. 对比已有的 `软件清单.xlsx`，不会删除旧记录
3. 新增的文件追加到表尾，已存在的跳过
4. 结果保存到当前文件夹的 `软件清单.xlsx`

## 依赖

- Python 3（`D:\anaconda3\python.exe`）
- openpyxl 库（`pip install openpyxl`）

## 常见问题

**Q: 双击后弹出安全警告？**  
A: 把这两个文件放到 C 盘用户目录下（如 `C:\Users\Administrator\`），C 盘一般不会被 Windows 拦截。

**Q: 磁盘上删掉的软件，表里的记录还在吗？**  
A: 在。脚本只追加不删除，历史记录永久保留。
